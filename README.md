# AmkyawDev - AI Power Platform

မြန်မာဘာသာစကားအထူးပံ့ပိုးပါဝင်သော AI Platform ဖြစ်ပါ့မယ်။

## အင်္ဂါရပ်များ

- 🗣️ မြန်မာဘာသာစကားအထူးပံ့ပိုး
- 💬 စာသားထုတ်လုပ်ခြင်း
- 🖼️ ပါမောက္ခချက် (DALL-E)
- 🎙️ အသံဖိုင်ပြောင်းခြင်း
- 🔊 စာသားအသံဖိုင်ပြောင်းခြင်း
- 🌐 ဘာသာပြန်ခြင်း
- 📝 အကျဉ်းချုပ်ခြင်း

## မော်ဒယ်များ

- GPT-4
- GPT-3.5 Turbo
- Claude 3
- Llama 3

## စတင်အသုံးပါဝင်နည်း

```bash
# Install dependencies
npm install

# Start server
npm start
```
